Project Overview

A comprehensive three-tier scraping architecture designed to extract critical property listing data from 99acres.com. This project solves the most common challenges in real estate scraping, including anti-bot detection, dynamic class randomization, and Cloudflare protection.

Three-Tier Architecture

Version 1: Semantic Engine

Bypasses daily class randomization by using semantic text detection. Finds property cards by identifying price symbols and BHK patterns directly on the render.

Version 2: Deep Crawler

Ensures 100% data integrity by depth-crawling individual property pages. Extracts full technical specs and direct broker information.

Version 3: Parallel HTTP

Enterprise-scale performance. Utilizes asynchronous HTTP requests and ScraperAPI residential proxies to fetch hundreds of properties in seconds.

Security & Stealth

Integrated playwright-stealth, header rotation, and automated TLS fingerprinting to ensure long-term reliability and ban prevention.
